このサイトでは、Oracle Database を題材にDB セキュリティの基本〜実践を、手順ベースで学べるようにまとめています。  
まずはこの「環境のセットアップ」で、以降のチュートリアル（認証／MFA／アクセス制御など）を進めるための環境を用意します。

手順は上から順に進める想定です。途中でつまずいた場合も、いったんこのセクションに戻って環境が揃っているか確認してください。  
なお、掲載しているコマンド出力やログは読みやすさのために整形・省略している場合があります。
